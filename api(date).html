Kas ir API

Kas ir API vienkāršā izteiksmē?

API – Application Programming Interface,

kas nozīmē lietojumprogrammu saskarni. API kontekstā vārds "lietojumprogramma" attiecas uz jebkuru programmatūras daļu ar noteiktu funkciju.

Interfeisu var uzskatīt par pakalpojumu līgumu starp divām lietojumprogrammām.
